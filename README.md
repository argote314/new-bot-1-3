## Clonar este proyecto

```bash
> git clone https://github.com/argote314/nuevo-bot-13-1
```

## Instale las dependencias:
Antes de ejecutar el siguiente comando, asegúrese de estar en el directorio del proyecto que
acabas de clonar !!
```bash
> cd nuevo-bot-13-1
> bash install.sh
```

### Activar
```bash
> npm start
```
